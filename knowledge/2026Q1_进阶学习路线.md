# Yoyoo 进阶学习路线（2026Q1）

> 目标：不是“学更多”，而是让 Yoyoo 更稳、更准、更省钱、更可复制。

## A. 稳定性与成本（先做）

1. OpenClaw 记忆与 QMD
- 学什么：`memory.backend=qmd`、检索范围、超时和注入长度限制。
- 价值：降低上下文膨胀，提升长会话稳定性。
- 资料：
  - https://docs.openclaw.ai/concepts/memory

2. LiteLLM 路由与故障切换
- 学什么：多模型路由、fallback、超时与重试。
- 价值：单模型波动时自动兜底，避免“员工宕机”。
- 资料：
  - https://docs.litellm.ai/
  - https://docs.litellm.ai/docs/routing
  - https://docs.litellm.ai/docs/proxy/reliability

## B. 可观测与可验收（第二优先）

3. Langfuse 可观测
- 学什么：trace、成本、失败链路、prompt 版本。
- 价值：把“感觉卡”变成“有证据可定位”。
- 资料：
  - https://langfuse.com/

4. Promptfoo 评测与红队
- 学什么：回归评测、CI 质量门、红队扫描。
- 价值：版本升级不再靠主观体验，改动可量化。
- 资料：
  - https://github.com/promptfoo/promptfoo
  - https://www.promptfoo.dev/docs/integrations/ci-cd/

## C. 大脑设计与安全治理（并行推进）

5. Letta 记忆块建模
- 学什么：persona/human/planning 等 memory blocks，跨 agent 共享块。
- 价值：为“CEO+部门员工”组织结构提供可编排记忆模型。
- 资料：
  - https://docs.letta.com/guides/agents/memory-blocks/
  - https://docs.letta.com/guides/agents/base-tools/

6. Agent 安全基线（OWASP + NIST）
- 学什么：Prompt Injection、过度授权、插件供应链风险。
- 价值：降低 Skill/插件带来的系统级风险。
- 资料：
  - https://genai.owasp.org/llm-top-10/
  - https://www.nist.gov/publications/artificial-intelligence-risk-management-framework-generative-artificial-intelligence

## D. 本周落地清单（建议）

- Day 1：完成 QMD 参数复核 + LiteLLM fallback 演练。
- Day 2：接入 Langfuse 并打通一条完整 trace。
- Day 3：为 3 个关键任务编写 Promptfoo 评测用例。
- Day 4：定义 Yoyoo memory blocks（CEO/后勤/研发最小集合）。
- Day 5：按 OWASP Top10 做一次技能安装安全审计。
