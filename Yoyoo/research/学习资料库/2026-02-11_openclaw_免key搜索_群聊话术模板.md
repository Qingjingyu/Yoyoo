# OpenClaw 免 Key 搜索群聊话术模板

- 日期：2026-02-11
- 场景：你在群里 @Yoyoo CEO 下发任务

## 1. 标准版（推荐）

```text
@YoyooCEO
执行“免Key搜索标准流程”：
1) 用 duckduckgo-search 搜索：<问题>
2) 选2个最相关链接，用浏览器打开并读取正文
3) 输出：
   - 结论（最多3条）
   - 证据（链接+一句话）
   - 不确定项（如有）
   - 下一步建议（1-2条）
要求：不要编造，查不到就明确写“未检索到充分证据”。
```

## 2. 快速版（短消息）

```text
@YoyooCEO 按“搜索->开网页->摘要”流程回答：
问题：<问题>
输出：结论3条内 + 证据链接 + 下一步建议。
```

## 3. 对比评测版（让 CEO 与后勤一起做）

```text
@YoyooCEO @Yoyoo后勤
同题并行检索：<问题>
每人都按“免Key搜索标准流程”输出。
最后由 CEO 汇总：
1) 共同结论
2) 分歧点
3) 建议采用方案
```

## 4. 失败重试话术

```text
如果本轮浏览器读取失败：
1) 先返回搜索结果前5条（标题+链接）
2) 给3个可替代关键词
3) 自动重试一次后再汇报
```
