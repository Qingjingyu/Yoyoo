# OpenClaw 四件事验收清单（安装完成后）

- 日期：2026-02-11
- 用途：5-10 分钟确认“已装、可用、可观测”

## A. 基础可用性

1. `openclaw --version` 有输出且版本正确  
2. `openclaw gateway status --deep` 显示运行中  
3. `openclaw logs --limit 100 --plain` 无连续致命错误

## B. 浏览器能力验收

命令：

```bash
openclaw browser --browser-profile chrome tabs
```

通过标准：
1. 能返回 tab 列表
2. 扩展状态可切换（`ON`）

## C. 技能生态验收

检查技能目录：

```bash
ls -la ~/.openclaw/skills
```

通过标准：
1. 已安装技能目录存在
2. 白名单 7 项技能都在（按实际 slug 检查）

## D. QMD 验收（降 token 核心）

1) 配置检查：

```bash
grep -n '\"backend\"' ~/.openclaw/openclaw.json
```

应包含：`"backend": "qmd"`

2) 运行日志检查：

```bash
openclaw logs --limit 200 --plain | grep -i qmd
```

应出现 qmd 相关运行日志。

## E. 指标验收（建议做 A/B）

至少回放同一批 20 个任务，对比：
1. 平均 token
2. p50 / p95 响应时延
3. 超时率 / 失败率
4. 单任务成本

达标建议：
1. 平均 token 下降 >= 60%
2. p50 时延下降 >= 40%
3. 失败率不高于切换前

## F. 交付判定

全部通过才算交付完成；任一项失败则按回滚手册处理。  
回滚文档：`2026-02-11_openclaw_四件事_回滚手册.md`
