# OpenClaw 7×24 AI 助理扫盲指南（安仔）

> **调研日期**: 2026-02-04
> **来源**: 安仔公众号长文
> **目标**: 总结 OpenClaw 成长故事、核心能力、部署方案对比与小白建议。

## 1. 故事与时间线
- 创始人 Peter（PSPDFKit 创始人，倦怠后被 Claude Code 点燃）。
- 2025-11 前身 Clawdbot（WhatsApp 中继）→ 拟人化/多任务/桌面化 → 2026-01 多平台 → 企业级特性（长记忆/多 Provider/安全）。

## 2. OpenClaw 是什么
- TS CLI 进程，24/7 在线 Agent 中控，可接入多聊天渠道（TG/WA/Slack/Discord/飞书等）。
- 调 LLM（Claude/GPT/本地），执行命令、管文件、控浏览器（语义快照）、可自装插件（Skills 700+）。
- Lane-based 队列：默认串行、显式并行，避免竞态，确保会话隔离。

## 3. 核心能力
1) **持久记忆**：会话转录 JSONL + MEMORY.md 摘要，自动写入与检索（向量+关键词）。
2) **能干活**：命令执行（容器/主机/远程），文件管理，Playwright 语义 DOM 快照（低成本）。
3) **自装插件**：Skills 生态 700+，可用自然语义自编自装。

## 4. 部署方案对比（成本/难度）
| 方案 | 适合人群 | 服务器成本 | API 成本 | 技术难度 | 部署时间 | 推荐 |
|------|----------|-----------|---------|---------|---------|------|
| 阿里云一键 | 完全小白、省心 | ¥30-50/月 | ¥20-50/月 | ⭐ | ~10 分钟 | ⭐⭐⭐⭐⭐ |
| VPS 自建 | 有基础、想省钱 | ¥0-50/月 | ¥70-210/月 | ⭐⭐⭐ | 30-60 分钟 | ⭐⭐⭐⭐ |

- 选择建议：
  - 预算高/零基础 → 阿里云一键（省事，有客服）。
  - 预算低/愿动手 → VPS（Oracle 免费/Hetzner 性价比）。
  - 有闲置机/隐私优先 → 本地。

## 5. 方案要点
- **阿里云一键**：预置镜像+百炼 API Key，一键放通 18789，10 分钟上线，成本 ~¥50-100/月。
- **VPS**：Ubuntu + Docker Compose；4GB RAM 常用；Oracle A1 免费/Hetzner €3-4/月；注意国际网络连通性，必要时代理。
- **本地**：npm -g openclaw；openclaw onboard；可配 Cloudflare Tunnel 远程访问；需 24/7 运行与安全防护。

## 6. 成本与网络
- LLM 轻度用量：$10-30/月（¥70-210）。
- 流量低，CPU/RAM 4GB 足够；需能直连 Anthropic/OpenAI/聊天平台，地区受限需代理。

## 7. 安全/风险提示
- 不在主力机部署，不裸露端口；最小权限 token；谨慎第三方 Skills。
- 控制成本：限制 token，定期清理会话/记忆。
- 渐进放权：先低风险任务，熟悉后再管重要数据/高风险操作。

## 8. 对 Yoyoo 的参考
- 结合我们的 AI 员工化/SOP：选 VPS 路线（性价比+可控），配三层记忆与审计。
- 加强安全：最小 token、隔离容器、日志与回滚；网络需保国际连通性。
- 成本治理：模型路由与 token 限额，定期归档记忆。
