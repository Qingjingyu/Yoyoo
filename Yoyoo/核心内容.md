## 已安装的保护措施

### 1. Fail2ban
- 保护 SSH 免受暴力破解
- 5 次失败尝试封禁 1 小时

### 2. Swap 空间
- 4GB Swap 文件
- 防止内存耗尽

### 3. 服务自动重启
- systemd 托管 yoyoo-api 服务
- 崩溃后 5 秒自动重启
- 内存限制: 1GB

### 4. 定时健康检查
- 每天凌晨 3 点检查
- 内存 > 90% 或磁盘 > 80% 自动重启服务

## 故障排除

### SSH 连不上
1. 检查网络连接
2. 使用阿里云 VNC 查看状态
3. 通过 Cockpit 重启 SSH: `systemctl restart ssh`

### API 连不上
```bash
# 检查服务状态
systemctl status yoyoo-api

# 重启服务
systemctl restart yoyoo-api

# 查看日志
journalctl -u yoyoo-api -n 50
```

### 内存/磁盘满
1. 清理日志: `journalctl --vacuum-time=7d`
2. 重启服务释放内存: `systemctl restart yoyoo-api`
3. 查看磁盘: `df -h`
